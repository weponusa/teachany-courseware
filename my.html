⚙️ 备份与清理

📤 导出全部历史

一键将本机所有学习记录导出为 JSON 文件，便于跨设备迁移或定期备份。文件中包含 6 个数据桶（views / quizzes / tutor_chats / pbl_runs / created）。

📥 导入备份

从之前导出的 JSON 文件恢复历史记录。"合并"模式会保留现有数据并加入新条目；"覆盖"模式会先清空再写入。

点击或拖放 JSON 文件到此

🗑️ 清空某个 Tab

仅清除选定数据桶，其他记录保留。

⚠️ 清空全部

不可撤销！建议先导出备份，再清空全部本地学习记录。
