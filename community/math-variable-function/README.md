# 变量与函数 (Variables and Functions)

**学科**：数学 | **年级**：七年级 | **知识点**：variable-and-function

## 课件简介

本课件通过"自动售货机"类比、互动判断卡片、三种表示法切换等方式，帮助学生深入理解变量与函数的核心概念。

## 教学设计亮点

- 🤖 **自动售货机互动**：直觉化理解函数的"唯一对应"本质
- 🃏 **判断卡片互动**：4 种典型对应关系，即点即知是否为函数
- 📊 **三种表示法切换**：解析式/列表/图像，一键对比
- 🧠 **深层理解模块**：三个认知镜头，突破易错概念
- 📈 **函数图像画板**：Canvas 实时渲染 y = 2x+1

## 知识结构

```
有理数运算 ──┐
              ├──→ 变量与函数 ──→ 坐标系
代数式     ──┘         └──→ 正比例函数 ──→ 一次函数
```
