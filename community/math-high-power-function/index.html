📖 模块一：幂函数的定义

定义

一般地，函数 y = xα 叫做幂函数，其中 x 是自变量，α 是常数（α ∈ R）。

y = xα，α ∈ R

注意要点

  • 幂函数的底数是自变量 x，指数是常数 α
  • 系数必须为 1（y = 2x² 不是幂函数）
  • 没有常数项（y = x² + 1 不是幂函数）

与指数函数的区别

函数类型一般形式自变量位置例子
幂函数y = xα底数y = x²
指数函数y = ax指数y = 2x

图像特征记忆口诀

  • α > 0：图像过原点 (0,0) 和 (1,1)，在第一象限内递增
  • α < 0：图像不过原点，过 (1,1)，在第一象限内递减
  • α 为偶数：图像关于 y 轴对称（偶函数）
  • α 为奇数：图像关于原点对称（奇函数）
📈 模块三：幂函数的单调性

单调性规律

对于幂函数 y = xα：

  • 当 α > 0 时，函数在 (0, +∞) 上单调递增
  • 当 α < 0 时，函数在 (0, +∞) 上单调递减

比较大小的方法

比较幂函数值的常用方法：

  1. 看单调性：若 α > 0，指数相同，底数大的函数值大（在正数范围内）
  2. 看奇偶性：利用对称性将负数转化到正数区间
  3. 找中间值：如与 0、1 比较
例：比较 1.50.3 与 1.20.3

∵ α = 0.3 > 0，y = x0.3 在 (0,+∞) 递增

∴ 1.50.3 > 1.20.3

🔍 模块四：幂函数的奇偶性

判断方法

对于幂函数 y = xα，其奇偶性由指数 α 决定：

α 的特征奇偶性举例对称性
α 为奇数奇函数y = x, x³, x⁻¹关于原点对称
α 为偶数偶函数y = x², x⁻², x⁴关于 y 轴对称
α 为非整数需具体分析y = x½定义域不对称，非奇非偶

奇偶性的应用

  1. 简化函数值计算：f(-x) = -f(x) 或 f(-x) = f(x)
  2. 判断图像对称性，辅助作图
  3. 比较异号自变量函数值的大小
例：判断 f(x) = x3 的奇偶性

f(-x) = (-x)³ = -x³ = -f(x)

∴ f(x) = x³ 是奇函数

📋 知识小结

核心知识点

  1. 定义：y = xα（α 为常数），系数为1，无常数项
  2. 五种典型：α = -2, -1, 1, 2, 3，掌握其图像与性质
  3. 单调性：α > 0 时在 (0,+∞) 递增；α < 0 时递减
  4. 奇偶性：α 为奇数→奇函数；α 为偶数→偶函数

易错点

  • ⚠️ 幂函数与指数函数形式易混淆（注意自变量位置）
  • ⚠️ y = x0 的定义域为 x ≠ 0（0⁰ 无意义）
  • ⚠️ 比较大小时要先确定函数的单调性区间
