推和拉的力

力是物理世界最基本的概念之一。今天我们认识两种最常见的力——推力和拉力，了解它们的方向、效果和大小关系。

👉 推力 🤏 拉力 💥 力的效果 📏 力的大小

LEARNING GOALS

📌 学习目标

🎯
能说出推力和拉力的定义，区分两种力的方向。
🔬
举例说明推力和拉力在生活中的应用（各≥3个）。
⚡
描述力的两个效果：改变运动状态 & 改变形状。
📏
理解力的大小与效果的关系：力越大效果越明显。

ABT NARRATIVE · 情境引入

🎬 一起来想一想

小明每天上学，都要推开校门进去，放学又要拉开柜子取书包。 他觉得这只是普通的生活动作——And（并且），他每天都这样做，习以为常。

But（但是），有一天科学老师问： "推门和拉柜子，用的力有什么不同？为什么有时候推一下就够，有时候要使劲推？" 小明愣住了——他从没想过这个问题！

Therefore（因此），今天我们就来探索这两种最基本的力： 推力和拉力。搞清楚它们的方向、效果和大小， 你也能像科学家一样理解生活中的每一个动作！

CORE CONCEPTS

💪 认识推力和拉力

两种力方向相反，但都能让物体运动起来。

👉

推力

用力把物体推离自己，物体向推的方向运动。

🚪 推开门 🛒 推小车 ⚽ 踢球 📦 推箱子
方向：从自己 → 向外
特征：物体离自己越来越远
🤏

拉力

用力把物体拉向自己，物体向拉的方向运动。

🗄️ 拉抽屉 🪢 拉绳子 🪁 拉风筝 🌾 拔草
方向：从外面 → 向自己
特征：物体离自己越来越近

EFFECTS OF FORCE

💥 力的神奇效果

力不只会移动物体，还能改变它的形状！

🏃

效果一：改变运动状态

  • 静止的物体 → 推/拉一下 → 动起来
  • 运动的物体 → 推/拉反向 → 慢下来/停住
  • 运动的物体 → 推/拉侧面 → 方向改变
🏺

效果二：改变物体形状

  • 捏橡皮泥 → 形状变了
  • 压弹簧 → 弹簧变短了
  • 拉橡皮筋 → 橡皮筋变长了

FORCE MAGNITUDE

📏 力的大小很重要！

同一方向的力，大一点效果就明显一点。

🐥
轻物体
用小力就能推动
移动距离较远
⚖️
🪨
重物体
需要大力才能推动
同等力下移动更少
💡 规律：力越大 → 运动越快 / 形变越大；物体越重 → 需要力越大

Q2. 下面哪个选项描述了力的效果？

B. 力能改变物体的运动状态和形状

D. 力是一种神秘的魔法
Q3. 用一个更大的力推同一辆小车，会发生什么？

B. 小车立刻停止

D. 没有任何区别

A. 推力改变了物体的形状
B. 拉力改变了物体的运动状态
C. 拉力改变了物体的形状
D. 没有力的作用
