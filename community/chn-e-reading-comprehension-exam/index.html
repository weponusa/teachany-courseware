第一关：8大题型图鉴
🗺️ 题型地图

阅读理解8大常考题型速览

AND你已经能读懂一篇文章
BUT面对各种不同类型的题目，不知道该用什么方法回答
THEREFORE我们把题目分类，每种题型配一个"解题公式"
📝
①概括类
谁 + 做了什么 + 结果怎样
（或：文章主要讲……，表达……）
🔍
②词语解释
联系上下文 → 本义 + 文中义
（"XX"在文中是指……）
💬
③句子含义
字面义 + 深层义（中心思想/情感）
（联系全文背景解释）
🎨
④修辞赏析
修辞名称 + 分析 + 效果
（生动形象地写出了……）
❤️
⑤情感主旨
文章表达了……的情感
说明了……的道理（记住：言之有据）
📦
⑥结构分析
总-分-总 / 分-总 / 总-分
（点明结构 + 说明作用）
🎭
⑦人物分析
外貌+动作+语言+心理描写
→ 人物品质/性格特点
✨
⑧开放题
表明观点 + 联系文章 + 联系生活
（言之有理，有据即可得分）

⚡ 方法特训

三种最难题型的解法

🔍 词语理解三步走
  1. 读上下文：找出词语所在的句子，结合前后文意思
  2. 本义+文中义：先写字典意思，再写在文中的具体含义
  3. 作用：这个词用在这里有什么效果（表现了什么？）
📋 词语解释答题框架
"XX"在文中的意思是：字面义+结合上下文+文中具体指， 表现了人物/环境特点。
💡 句子含义三层解析
第一层（表面）：句子表面的意思是什么？
第二层（语境）：联系上下文，这句话暗示了什么？
第三层（主旨）：这句话与文章中心有什么关系？

📄 阅读材料：《走一步，再走一步》（改编）
小亨特和几个朋友爬上了一个高悬崖，当他回头望去，一种说不出的恐惧和虚弱感袭来，他的双腿颤抖，无法动弹。父亲出现了，用平静的声音说："不要想着下面有多远，只想着下一步，就这一步。"亨特慢慢地，一步一步地爬了下来。当脚踩到地面的那一刻，他突然感到自己长大了。

1. 文中加下划线的"恐惧"在这里的含义是？

A 恐惧就是害怕，没有特别的意思。
B 指面对高处危险时的极度紧张与无助，突出了亨特因没有经验而陷入困境的心理状态。
C 表示亨特因为身体不好而感到害怕。

2. "他突然感到自己长大了"的深层含义是什么？

A 亨特比以前长高了，更强壮了。
B 亨特爬下了悬崖，说明他的身体变强了。
C 亨特通过克服困难，心理上成熟了——明白了面对挑战时要专注于当下的每一步，不必被困难的全貌压垮。

📄 限时阅读材料：《蚂蚁和蝈蝈》

炎热的夏天，蚂蚁们在烈日下搬运粮食，汗流浃背，脚步沉重，但没有一只停下来休息。蝈蝈们躲在树荫下，有的唱歌，有的嬉戏，笑话蚂蚁傻。

冬天来了，北风呼啸，大雪纷飞。蝈蝈们没有粮食，又冷又饿，后悔莫及。蚂蚁们却躲在暖暖的地下，吃着自己辛苦积累的粮食，过得舒舒服服。

1.（概括类）用一句话概括这篇文章的主要内容：

A 蚂蚁和蝈蝈是好朋友。
B 蚂蚁夏天勤劳储粮，蝈蝈嬉戏不备粮，冬天蝈蝈受苦，蚂蚁安然过冬。
C 夏天很热，蚂蚁很辛苦。

2.（词语类）"后悔莫及"的意思是：

A 蝈蝈觉得自己很可怜
B 后悔已经来不及了，形容蝈蝈因为夏天不勤劳，冬天缺粮而深感懊悔
C 蝈蝈想找蚂蚁帮忙

3.（情感主旨）这篇文章说明了什么道理？

A 蚂蚁比蝈蝈聪明
B 只有居安思危、勤劳积累，才能在困难来临时从容应对
C 夏天要多吃东西
D 蝈蝈应该向蚂蚁道歉

4.（开放题）如果你是蝈蝈，回想夏天的选择，你会怎么想？（联系生活，2-3句话）

📄 阅读材料：《母亲的手》

我记得小时候，母亲的手总是那么温暖，粗糙而有力。每当我生病发烧，那双手就会轻轻地覆在我的额头上，像最好的药，让我的恐慌一点点消散。

后来我长大了，外出求学，很少回家。有一次回去，握住母亲的手，才发现那双手已经布满了裂纹，像干涸的土地，失去了往日的温润。我心里有一块东西突然碎了。

"妈，你以后别干那么重的活了。"我说。母亲笑笑："有什么关系，能干活，说明还健康呢。"那一刻，我突然觉得，母亲的手比以前更美了。

1.（概括）用一句话概括这篇文章的主要内容：（10分）

A 通过回忆母亲的手从温暖到粗糙的变化，表达了对母亲辛劳的心疼与爱。
B 我小时候生病，母亲照顾我。
C 母亲的手很粗糙。

2.（修辞）"那双手已经布满了裂纹，像干涸的土地"运用了什么修辞，有什么表达效果？（10分）

A 拟人，把手当作人来写，生动形象。
B 比喻，将母亲布满裂纹的手比作干涸的土地，生动形象地写出了母亲手部皮肤的粗糙与老化，突出了母亲多年劳累的辛苦，令人心疼。
C 夸张，夸大了手裂纹的样子。

3.（句子含义）"我心里有一块东西突然碎了"中"碎了"的含义是？（10分）

A 我摔碎了一件东西。
B 比喻心里某种感情（对母亲的心疼与愧疚）被触动，内心产生了强烈的震撼与难受，用"碎了"形象地写出了内心的痛。
C 我感到心脏出了问题。

4.（开放题）读完这篇文章，你想对你的父母说什么？联系生活实际，写2-3句话。（15分）
