# There be 句型（小学英语·外研社版）

## 基本信息

| 属性 | 值 |
|------|-----|
| 学科 | 英语 (English) |
| 年级 | 小学 3-4 年级 |
| 教材版本 | 外研社版 |
| 知识点 | There be 句型 |
| 难度 | ⭐⭐ (2/5) |
| 互动类型 | 选择题 / 填空 / 拖拽 / 开放式创作 |

## 课件简介

本课件围绕 There be 句型展开，通过 ABT 叙事引入、场景化 SVG 交互和分层练习，帮助小学生掌握：

1. **There is / There are** 的正确使用规则
2. **否定句** 和 **一般疑问句** 的转换
3. 用 there be 描述真实场景（教室/房间）

## 模块结构

| # | 模块 | 核心内容 |
|---|------|----------|
| 1 | 🎯 目标 & 前测 | 学习目标 + 前置知识检测 |
| 2 | 🌟 ABT 引入 | "This is..." → "There be..." 叙事转折 |
| 3 | 🔑 is/are 规则 + 练习 | 单数→is，复数→are（口诀记忆）+ 3道填空 |
| 4 | 🏫 教室实战 | SVG 教室场景拖拽填空（9个物品） |
| 5 | 🔄 变换 & 创作 | 否定/疑问句转换 + 开放式房间描述 |

## 技术特性

- 零依赖单文件 HTML（无需网络）
- 纯 SVG 内联场景图（教室 + 卧室）
- 支持触摸设备拖拽交互
- TeachAny 自适应学习引擎（Mastery / Progress / SR / Achievement）
- 响应式设计（适配手机/平板）

## 使用方式

直接用浏览器打开 `index.html` 即可开始学习。
