📖 核心概念

📏 什么是周长？

周长是围成一个图形的所有边的总长度。

周长 = 各条边之和
💡 "周"表示一圈，"长"表示长度。周长就是绕图形走一圈的距离。

🔲 长方形周长

长方形有两条长和两条宽。

C = 2 × (长 + 宽)
C = 2长 + 2宽

例：长6cm，宽4cm
周长 = 2×(6+4) = 20cm

⬛ 正方形周长

正方形四边相等。

C = 4 × 边长

🔺 三角形周长

三角形三边之和。

C = a + b + c

⬠ 多边形周长

各边长度之和。

C = 各边之和

🎯 后测：学会了吗？

完成学习后，用这两道题检验你的掌握程度！

Q1：长4cm宽3cm的长方形，周长是多少？

✅ 长方形周长=(长+宽)×2=(4+3)×2=14cm

Q2：边长5cm的正方形，周长是多少？

✅ 正方形周长=边长×4=5×4=20cm
🧠 记忆锚点

记忆口诀：长方形周长=(长+宽)×2；正方形周长=边长×4。助记：想象蚂蚁爬一圈边，走完所有边的总距离就是周长。口诀："长加宽，乘以二，长方周长记牢里；四边等长是正方，边长乘四是周长。"

🔎 错因诊断

⚠️ 如果你做错了，看看错在哪里：

如果你把周长公式写成(长+宽)而忘了×2，你忘了长方形有两组对边。每条边要算一次，长方形有两条长和两条宽，所以要×2。

💡 自查方法：把答案代回原题验证，如果算出矛盾就说明中间有错误！
