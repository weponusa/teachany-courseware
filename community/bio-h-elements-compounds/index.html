所以我们要理解碳的四键特性，以及碳链如何构建出生命的多样性。

🎬 碳原子特性 → 碳链形成 → 氨基酸脱水缩合 → 蛋白质折叠（共16秒，含语音解说）

⚛️ 碳元素的四键优势
碳原子结构特点
碳有 4个外层电子 → 可形成4个共价键 → 键合灵活，可链状、支链、环状
🔗
C-C 单键：可自由旋转，形成柔性长链（如脂肪酸）
🔄
C=C 双键：刚性，不能旋转（如不饱和脂肪酸）
⭕
碳环：苯环、葡萄糖六元环（提供稳定结构）

📖 模块 3：糖类——细胞的能源与结构材料

AND

运动前吃碳水（糖类）能快速补充能量……

BUT

淀粉、葡萄糖、纤维素都是糖类，但为什么有的能提供能量，有的不能？它们有什么区别？

THEREFORE

所以今天要学习糖类的分类（单糖/二糖/多糖）和各自的功能。

🍬 糖类三级分类
⚖️ 核心辨析：淀粉 vs 糖原 vs 纤维素
多糖 存在于 功能 单体
淀粉 植物细胞 储能（马铃薯/种子） α-葡萄糖
糖原 动物细胞 储能（肝脏/肌肉） α-葡萄糖
纤维素 植物细胞壁 结构支撑，不可消化 β-葡萄糖
🔑 易错点： 淀粉和糖原单体都是葡萄糖，但连接方式不同，人体淀粉酶不能分解纤维素！
✏️ 即时练习

下列哪项不是二糖？

✏️ 解释题（重要）

某同学说："脂肪和糖类都是储能物质，所以可以相互替代，减脂只需不吃糖就够了。"

请从结构角度分析这种说法是否正确，并说明理由。

参考答案：这种说法不完全正确。①糖类除储能外还有结构功能（纤维素构成细胞壁、核糖构成核酸），脂肪无法替代；②脑细胞几乎只能利用葡萄糖供能；③糖类氧化分解更快，提供能量更迅速；④脂肪的储能密度更高但不溶于水，不能溶解于血液直接运输。所以两者不能完全互相替代。

📖 模块 4：蛋白质——功能的执行者

AND

你知道蛋白质很重要，肌肉、酶、抗体都是蛋白质……

BUT

为什么这些蛋白质功能差异如此巨大？都是蛋白质，为什么胰岛素能降血糖，而血红蛋白能运氧气？

THEREFORE

所以要搞清楚蛋白质的结构层次和"结构决定功能"的核心原理。

🧬 蛋白质结构层次（全景图）
⚗️ 氨基酸的通用结构
氨基酸通用结构
    H
    |
H₂N-C-COOH
    |
    R（侧链，决定氨基酸种类）

🔑 每种氨基酸必须含有：氨基（-NH₂）和羧基（-COOH），且都连在同一个碳原子上

氨基酸的多样性来自 R 侧链
• R = H → 甘氨酸（最简单）
• R = CH₃ → 丙氨酸
• R含 -OH → 极性氨基酸
• R含 -SH → 半胱氨酸（形成二硫键）
• 共20种基本氨基酸
🔗 脱水缩合反应形成肽键
氨基酸₁
+
氨基酸₂
→
二肽 + H₂O
重要公式
n个氨基酸 → 1条多肽 + (n-1)个水分子，含(n-1)个肽键
⚠️ 易错点： 多条肽链组成的蛋白质中，游离氨基/羧基数 = 每条肽链两端 + R基中含有的数量。脱水数 = 肽键数，不等于总水分子数！

⭐⭐ 拓展选做题：蛋白质多样性分析

某蛋白质由300个氨基酸组成3条多肽链，请回答：

① 该蛋白质中最少含有多少个游离的氨基（只考虑肽链两端，不考虑R基）？

② 共形成多少个肽键？脱去多少个水分子？

③ 若蛋白质发生变性（空间结构改变），氨基酸序列改变了吗？

答案：
① 游离氨基最少3个（每条多肽链N端各一个，共3个；不含R基中的）
② 肽键数 = 300 - 3 = 297个；脱去水分子数同样 = 297个
③ 变性不改变氨基酸序列（一级结构不变），但改变了空间构型（二级/三级结构），导致功能丧失

📌 课堂小结

🗂️ 核心知识回顾
元素层面
  • C H O N P S 是主要元素
  • 含量最多：O；干重最多：C
  • 碳是有机物骨架核心
化合物层面
  • 水：含量最多的化合物
  • 蛋白质：含量最多的有机物
  • 糖类：能源与结构材料
糖类规律
  • 单糖（葡萄糖、核糖）不可再水解
  • 二糖（蔗糖、麦芽糖）水解得单糖
  • 多糖：淀粉（植）糖原（动）纤维素
蛋白质规律
  • 20种氨基酸，R基决定种类
  • n个氨基酸 → n-1个肽键 + n-1个水
  • 结构（序列+空间构型）决定功能
