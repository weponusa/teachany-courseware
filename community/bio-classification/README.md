# 生物分类 — 八年级生物互动课件

## 课件信息

- **学科**：生物
- **年级**：八年级（初中）
- **课时**：约 40 分钟
- **节点 ID**：`bio-m-bio-classification`
- **课标领域**：生物多样性与跨学科实践

## 课标对接

依据《义务教育生物学课程标准（2022 年版）》：

1. 根据生物之间的相似程度将生物划分为**界、门、纲、目、科、属、种**等分类等级
2. 说明生物的不同分类等级及其相互关系，初步形成**生物进化的观点**
3. 充分利用本地的生物资源，组织学生识别生物的特征，**尝试开展分类活动**

## 教学设计亮点

- **ABT 叙事**：从"全球已知生物 200 万种、每天还在发现新物种"切入，制造"没有共同语言无法交流"的认知冲突
- **Five-Lens 方法**：看见它（树状层级图）+ 拆开它（七级递进卡片）+ 比较它（家猫/华南虎/大熊猫亲缘对比）
- **探究式学习 L2**：本地物种亲缘判断活动（家猫 / 华南虎 / 大熊猫 / 狼 / 狗），学生选证据、做假设、验证
- **自适应分支**：前测后判断是否跳过基础模块；综合练习 Level 2 后判断是否加深迁移题
- **错因诊断反馈**：每道题针对错误选项给出具体认知错位说明（如"长得像就是同种"误区）
