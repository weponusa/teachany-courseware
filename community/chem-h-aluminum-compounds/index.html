❓引入：铝制易拉罐的秘密

生活中铝无处不在：易拉罐、铝箔纸、飞机机身、门窗框架。铝的密度只有铁的三分之一，却拥有不错的强度，而且表面总有一层亮闪闪的薄膜。

但有一个奇怪的现象：铝制易拉罐不能长时间装酸性饮料（如碳酸饮料）或碱性液体。 为什么？

更奇怪的是，铝既能与盐酸反应放出氢气，也能与氢氧化钠溶液反应放出氢气。大多数金属只与酸反应，铝为什么"两边都行"？

这就是我们今天要探究的核心问题。

⚛️核心概念一：铝单质的性质

1. 物理性质

银白色金属，密度 2.70 g/cm³（约为铁的 1/3），熔点 660°C，具有良好的导电性、导热性和延展性。

2. 化学性质——表面氧化膜的保护作用

铝是活泼金属，但常温下与空气接触会迅速形成一层致密的 Al₂O₃ 氧化膜（厚度约 4 nm），阻止内部铝继续被氧化。这层膜是铝制品耐腐蚀的关键。

3. 与酸反应

与盐酸反应 2Al + 6HCl → 2AlCl₃ + 3H₂↑

现象：产生大量无色气泡（氢气）。本质是 Al 失去电子，H⁺ 得到电子。

4. 与强碱反应

与氢氧化钠反应 2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑

现象：产生无色气泡。反应分两步：Al 先与水反应生成 Al(OH)₃ 和 H₂，生成的 Al(OH)₃ 再与 NaOH 反应生成可溶的 NaAlO₂，从而使反应持续进行。

🧪核心概念二：氧化铝的两性

氧化铝（Al₂O₃）是一种典型的两性氧化物：既能与酸反应生成盐和水，也能与强碱反应生成盐和水。

与酸反应（表现碱性）
Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O
与碱反应（表现酸性）
Al₂O₃ + 2NaOH → 2NaAlO₂ + H₂O

由于 Al₂O₃ 具有两性，铝制品在酸性或碱性环境中都会被腐蚀。这就是铝制易拉罐不宜长时间装碳酸饮料或碱性清洁剂的原因。

🔬核心概念三：氢氧化铝的两性 · 虚拟实验

氢氧化铝 Al(OH)₃ 是一种白色胶状沉淀，同样具有两性。下面的虚拟实验可以帮助你直观理解：向含有 Al³⁺ 的溶液中逐滴加入试剂，观察沉淀的生成与溶解。

实验原理与结论

制备 Al(OH)₃：Al³⁺ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄⁺（用弱碱氨水，避免过量强碱溶解沉淀）

与酸反应：Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O

与强碱反应：Al(OH)₃ + NaOH → NaAlO₂ + 2H₂O

结论：Al(OH)₃ 既能与酸反应，也能与强碱反应，是典型的两性氢氧化物。

🔥核心概念四：铝热反应

铝热反应是铝在高温下还原金属氧化物的反应，放出大量的热，温度可达 2500°C 以上。

铝热反应（以氧化铁为例） 2Al + Fe₂O₃ —高温→ 2Fe + Al₂O₃ + 大量热

该反应需要高温引发，通常用镁条和氯酸钾作为引燃剂。反应一旦开始，放出的热量足以维持反应继续进行。

工业应用
  • 焊接钢轨（现场修复铁路）
  • 冶炼难熔金属（如钒、铬、锰）
  • 火箭固体燃料助推
反应特点
  • 放热剧烈，温度极高
  • 产物铁呈熔融态
  • 需在安全环境下操作
  • 铝作为还原剂，体现还原性

✏️例题与演练

例题 1 写出铝与氢氧化钠溶液反应的离子方程式，并标出电子转移方向和数目。

答案：2Al + 2OH⁻ + 2H₂O → 2AlO₂⁻ + 3H₂↑
电子转移：Al 失去 6e⁻（0 → +3），H⁺ 得到 6e⁻（+1 → 0）

例题 2 为什么制备 Al(OH)₃ 时常用氨水而不用 NaOH 溶液？

答案：因为 Al(OH)₃ 具有两性，能与过量强碱（NaOH）反应生成可溶的 NaAlO₂，导致沉淀溶解。氨水是弱碱，即使过量也不会溶解 Al(OH)₃ 沉淀。
💡互动练习

判断以下说法是否正确：

  1. 铝在空气中不能燃烧，因为表面有氧化膜保护。（√ 正确）
  2. Al₂O₃ 既能与盐酸反应，也能与 NaOH 反应，所以它是碱性氧化物。（× 错误，是两性氧化物）
  3. 铝热反应中，铝作氧化剂，Fe₂O₃ 作还原剂。（× 错误，铝是还原剂）
  4. 向 AlCl₃ 溶液中逐滴加入 NaOH 溶液，先产生白色沉淀，后沉淀溶解。（√ 正确）
📝课堂小测

1. 下列关于铝的说法正确的是（ ）

  • A铝不能与碱反应
  • B铝表面有致密氧化膜，具有抗腐蚀性
  • C铝热反应中铝是氧化剂
  • DAl(OH)₃ 只能与酸反应
铝表面形成致密 Al₂O₃ 膜，阻止进一步氧化，这是铝抗腐蚀的原因。铝能与强碱反应；铝热反应中铝失电子作还原剂；Al(OH)₃ 是两性氢氧化物。

2. 实验室制备 Al(OH)₃ 的最佳试剂是（ ）

  • A过量 NaOH 溶液
  • B过量盐酸
  • C适量氨水
  • D过量稀硫酸
氨水是弱碱，能使 Al³⁺ 沉淀为 Al(OH)₃，且过量不会溶解沉淀。NaOH 过量会使沉淀溶解生成 NaAlO₂。

3. 铝热反应可用于焊接钢轨，其原理是（ ）

  • A反应放出大量热，使铁熔化流入缝隙
  • B反应生成铝蒸气填充缝隙
  • C反应产生高压将钢轨压合
  • D铝比铁活泼，置换反应自动进行
铝热反应放出大量热（约 2500°C），生成的铁呈熔融态，可流入钢轨缝隙冷却后形成牢固焊缝。
📋总结

铝单质

活泼金属，表面有致密 Al₂O₃ 保护膜。既能与酸反应，也能与强碱反应放出 H₂。

两性化合物

Al₂O₃ 和 Al(OH)₃ 都是两性物质：既能与酸反应，也能与强碱反应。

铝热反应

高温下放热反应，用于焊接钢轨和冶炼难熔金属。铝作还原剂。

核心线索：铝的"两面性"源于 Al³⁺ 处于金属与非金属的交界，使其化合物具有独特的两性特征。
