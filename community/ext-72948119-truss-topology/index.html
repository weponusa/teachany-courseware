Concept

什么是桁架？

桁架是由直杆在端点铰接而成的结构。每根杆件只承受轴向力（沿杆方向的拉力或压力），不承受弯矩。

✅ 三角形桁架

❌ 方形框架

三角形稳定性定理：三条边长唯一确定一个三角形 → 不会发生几何变形

⚠️ 易错点

"三角形稳定"≠ 三角形不会断。稳定性指几何构型不变（不变形），不是指材料不破坏。纸条可能被拉断，但三角形的形状不会变。

Concept

二力杆与节点平衡

二力杆：桁架中的杆件只在两端受铰接约束，因此力只能沿杆轴线方向传递。

← → 拉力 (T) 杆件被"拉长"
→ ← 压力 (C) 杆件被"压缩"

📝 节点法三步骤

  1. 取一个节点为隔离体，画受力图
  2. 对 x、y 方向分别列平衡方程：ΣFx = 0，ΣFy = 0
  3. 解方程求杆力（正值=拉力，负值=压力）
节点平衡：ΣFx = 0   ΣFy = 0  （每个节点两个方程）
Example

例题：用节点法求解三角桁架

纸桥 P1 的一个三角单元，顶点受 100 N 竖直载荷，两底脚铰支。求三根杆力。

📐 解题过程

  1. 取顶点 A为隔离体，外力 P=100N↓
  2. 设 AB 杆拉力 FAB，AC 杆拉力 FAC（等腰三角形，对称结构）
  3. 列 y 方向平衡：2FAB·sin60° = 100 → FAB = 100/(2×0.866) ≈ 57.7 N（拉力）
  4. 列 x 方向平衡：FAB·cos60° − FAC·cos60° = 0 → FAC = FAB（对称，必然相等）
  5. 底杆 BC：取 B 点，FAB·cos60° = FBC → FBC = 57.7×0.5 ≈ 28.9 N（拉力）
结果：AB=AC=57.7N（拉） BC=28.9N（拉） → 所有杆件受拉，纸条正好发挥抗拉优势！

💡 项目关联：P1 三角桁架纸桥中，每根纸条都受拉力，这是纸的最佳受力模式。后续加载测试中注意观察——失效通常发生在节点连接处（胶水/折缝），而非杆件中段断裂。

Extension

拓扑优化入门：少即是多

结构拓扑优化的核心问题：给定载荷和约束，用最少的材料布置最强的结构。

🔍 纸桥项目中的拓扑优化思维

  • 约束：纸张叠层数固定（控制变量）
  • 变量：结构形式（三角/拱/蜂窝）
  • 目标：承载效率 = 最大承载力 / 用纸质量
  • 思路：固定材料总量，改变力的传递路径，比较承载效率
承载效率 η = Pmax / m   （N/g）
Pmax = 最大承载力，m = 用纸质量

这就是你在加载测试中要计算和比较的核心指标。

4. 简单桁架中，节点受竖直向下载荷 P=200N，两杆与水平面夹角均为 45°。求每根杆的内力。

解：对节点列 y 方向平衡
  ΣFy = 0 → 2 × F × sin45° = P
  F = P / (2 × sin45°) = 200 / (2 × 0.707) ≈ 141.4 N（两杆均为压力）

Exercise Review

习题讲解：常见错误与正确解法

❌ 典型错误 1：混淆"稳定"和"强度"

错因：把"三角形不变形"理解成"三角形不会断"。纠正：稳定=几何构型不变；强度=材料不断裂。纸条可能被拉断（强度不足），但三角形的形状不会变（稳定）。

❌ 典型错误 2：节点法中力的方向设错

错因：画受力图时，把杆对节点的力方向画反（节点法中应假设杆力为拉力，指向远离节点）。纠正：始终设杆力为拉力→指向外。算出正=拉力，负=压力。

❌ 典型错误 3：只比最大承载力，忽略自重

错因：拱形纸桥可能承重更大，但用纸量也多。纠正：必须用承载效率 η = Pmax/m 比较，而非只看 Pmax。这正是你项目交付物中需要计算的。

✅ 正确解题模板

  1. 画桁架几何图，标注节点字母（A, B, C...）
  2. 取未知力≤2 的节点为起点，画隔离体受力图
  3. 假设所有杆力为拉力（箭头指向远离节点）
  4. 列 ΣFx=0 和 ΣFy=0，解方程
  5. 正号=拉力，负号=压力。在纸桥上标注方向

Summary

总结与迁移

1 三角形是基本稳定单元——三条边唯一确定几何形状，无需对角杆
2 二力杆只受轴向拉/压——节点法（ΣFx=0, ΣFy=0）可逐节点求解
3 三角桁架最适合纸桥——纸条以轴向力传载，发挥纸的抗拉优势
4 拓扑优化 = 固定材料→改变结构→比较承载效率 η=Pmax/m
