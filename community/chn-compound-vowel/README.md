# 复韵母乐园 · Compound Vowel Playground

> Subject: Chinese Language · Grade: 1 · Type: New Concept · Language: Chinese

## Overview

An interactive courseware for learning Chinese compound vowels (复韵母). Designed for Grade 1 students with a bright, colorful candy-themed UI following the TeachAny Elementary design system.

## Features

- 🌈 Colorful candy-themed design for young learners
- 🎯 Interactive compound vowel recognition exercises
- 🔊 Audio pronunciation support
- 📝 Writing practice with stroke guidance
- 🎮 Gamified learning experience

## How to Use

Open `index.html` in any modern browser. No build step or dependencies required.
