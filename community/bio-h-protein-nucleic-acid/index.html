📖 模块1：氨基酸——蛋白质的单体

🚀 情境引入

想象你在厨房里，手边只有 20种积木（氨基酸）， 却可以拼出几乎无限种形状。大自然就是这样用20种氨基酸，创造出数万种不同的蛋白质！

核心问题：这20种氨基酸有什么共同特征？又是什么让它们"各不相同"？

🔬 互动实验：探索氨基酸结构
👆 点击画布中的功能团标签来了解各基团的作用 | 点击"显示不同R基"按钮查看20种氨基酸的差异

🧪 氨基酸通用结构特点

  • 含有至少 一个氨基（-NH₂）
  • 含有至少 一个羧基（-COOH）
  • 氨基和羧基连接在 同一个碳原子（α-C） 上
  • R基（侧链） 决定氨基酸的种类
NH₂ — C(H)(R) — COOH
α-氨基酸通用结构式

⚠️ 易混点辨析

  • 部分氨基酸含多个氨基 / 羧基
  • 判断是氨基酸的关键：至少1个-NH₂ 和 1个-COOH 连在同一个碳上
  • R基不同 → 氨基酸种类不同
  • R基可以是氢原子（最简单的甘氨酸）
🔺 记忆口诀： 一氨基、一羧基、同碳连，R基变身显身份

📖 模块2：蛋白质——生命活动的主要承担者

🎬 动画演示：脱水缩合反应

观看蛋白质是如何由氨基酸"手牵手"形成的——每次"握手"都会脱去一分子水，形成肽键。

🎬 氨基酸→多肽链→蛋白质折叠过程动画 (含讲解音频)

⚗️ 脱水缩合反应

两个氨基酸分子脱去一分子水（H₂O），形成一个肽键（-CO-NH-）。

氨基酸₁-NH₂ + HOOC-氨基酸₂
↓ 脱H₂O
氨基酸₁-CO-NH-氨基酸₂ + H₂O
  • 形成1个肽键 = 脱去1分子水
  • 二肽：2个氨基酸，1个肽键
  • 三肽：3个氨基酸，2个肽键
  • n肽（n条）：肽键 = aa数 - n

📊 计算公式（高频考点）

单条肽链（n个氨基酸）：

肽键数 = n - 1
脱水数 = n - 1
氨基（游离）= 各aa氨基总和 - (n-1)

多条肽链（共m条）：

肽键数 = 氨基酸总数 - m
脱水数 = 氨基酸总数 - m
💡 相对分子质量 = 各aa质量之和 - 18 × 脱水数

🌀 蛋白质的结构层次

一级结构

氨基酸的排列顺序（肽键连接），是蛋白质结构的基础。氨基酸序列不同 → 蛋白质不同。

二级结构

α螺旋 / β折叠（局部空间结构），由氢键维持。

三级结构

整条肽链的三维空间构象，由各种非共价键维持。
功能活性的基础！

四级结构

多条肽链组合（如血红蛋白 = 4条肽链）。

🔑 蛋白质的功能多样性

⚡ 催化功能

酶（如消化酶、DNA聚合酶）催化细胞内化学反应，提高反应速率。

📡 调节功能

激素（如胰岛素、生长激素）调节细胞代谢和发育。

🛡️ 免疫功能

抗体（免疫球蛋白）识别并消灭入侵的病原体。

🚛 运输功能

载体蛋白（如血红蛋白、通道蛋白）转运物质。

🏗️ 结构功能

构成细胞膜、细胞骨架（如肌动蛋白、胶原蛋白）。

🔌 信号传导

受体蛋白（如信号受体）接收外界信号并传导至细胞内。

📖 模块3：核酸——遗传信息的载体

🌟 情境引入

你的身体里有 37万亿个细胞，每个细胞的细胞核里都藏着 约6000个碱基对的遗传密码（DNA）。这些DNA如果展开连接，可以绕地球3000圈！ 更神奇的是，这份遗传蓝图控制着你的每一个特征——眼睛的颜色、血型、甚至疾病风险。

核心问题：核酸是怎样储存和传递这份遗传密码的？

🧬 DNA（脱氧核糖核酸）

  • 基本单位： 脱氧核苷酸
  • 五碳糖： 脱氧核糖
  • 碱基： A(腺嘌呤)、T(胸腺嘧啶)、G(鸟嘌呤)、C(胞嘧啶)
  • 结构： 双链螺旋
  • 位置： 主要在细胞核，少量在线粒体/叶绿体
  • 功能： 储存遗传信息

📋 RNA（核糖核酸）

  • 基本单位： 核糖核苷酸
  • 五碳糖： 核糖
  • 碱基： A(腺嘌呤)、U(尿嘧啶)、G(鸟嘌呤)、C(胞嘧啶)
  • 结构： 单链
  • 位置： 主要在细胞质
  • 功能： 传递/翻译遗传信息

📖 模块4：DNA指导蛋白质合成——中心法则

💡 核心理解

蛋白质的结构（氨基酸序列）最终由DNA中的碱基序列决定。这个"信息流"叫做中心法则。

以DNA为模板，合成mRNA。在细胞核中进行（以DNA为模板）。
碱基对应：A→U, T→A, G→C, C→G

翻译（Translation）

以mRNA为模板，在核糖体上合成蛋白质。每3个碱基（密码子）对应1个氨基酸。
64个密码子 → 20种氨基酸

🏆 为什么说DNA是主要的遗传物质？

  • 绝大多数生物（动物、植物、真菌）的遗传物质都是DNA
  • DNA储存在细胞核中，相对稳定，不易受外界影响
  • DNA的碱基序列可以精确地复制传递给后代（DNA复制）
  • 例外：某些病毒（如HIV、流感病毒）遗传物质是RNA → 因此RNA也是遗传物质
📌 考点： 凡有细胞结构的生物，遗传物质都是DNA。有细胞结构 → DNA是唯一遗传物质。 无细胞结构的病毒，遗传物质可能是DNA也可能是RNA。

✏️ 即时练习 4-1（深层理解）

关于"中心法则"，下列说法正确的是：
✅ C 正确。中心法则：DNA→RNA（转录）→蛋白质（翻译）。 A：DNA不能直接翻译，需先转录为mRNA。B：部分RNA病毒通过逆转录酶可以RNA→DNA。 D：信息不能从蛋白质逆向流回DNA（蛋白质序列不能逆转为核酸序列）。

⭐ 基础巩固（必做）
一个蛋白质分子有3条肽链，总共包含200个氨基酸。请问：
① 该蛋白质含有多少个肽键？
② 形成该蛋白质时脱去了多少分子水？
③ 若每个氨基酸平均相对分子质量为128，则该蛋白质的相对分子质量约为多少？
📝 解答：

① 肽键数 = 氨基酸总数 - 肽链条数 = 200 - 3 = 197 个肽键

② 脱水数 = 氨基酸总数 - 肽链条数 = 200 - 3 = 197 分子水

③ 蛋白质相对分子质量 = 各aa质量总和 - 18 × 脱水数
= 200 × 128 - 18 × 197 = 25600 - 3546 = 22054
⭐⭐ 拓展应用（选做）
某DNA分子有1000个碱基对，其中腺嘌呤（A）占碱基总数的30%。
① DNA分子中，胸腺嘧啶（T）有多少个？
② 鸟嘌呤（G）有多少个？
③ 若用此DNA转录出mRNA，该mRNA中尿嘧啶（U）的碱基组成比例是多少？（假设两条链各转录一次的一半）
📝 解答：

DNA总碱基数 = 1000 × 2 = 2000 个

① A = T（互补配对），A占30%，即A = 2000 × 30% = 600个，T = 600个

② G = C，且 A+T+G+C = 2000，所以 G+C = 2000 - 1200 = 800，G = 400个

③ mRNA中U的比例需要知道模板链。转录时A→U，所以mRNA的U来自模板链的A。 若以含A比例高的链为模板，则mRNA中U比例更高，实际考题中通常指定模板链。 （此题③为开放性，实际答题时需知道是哪条链为模板）

  • 碱基互补：A-T（2键），G-C（3键）
  • 中心法则：DNA→RNA→蛋白质
