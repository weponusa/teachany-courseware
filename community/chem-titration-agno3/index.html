And → But → Therefore

为什么需要滴定法？

And：已知 AgNO₃ 遇 Cl⁻ 产生白色沉淀 AgCl——定性分析。

But：生活需定量答案：生理盐水 NaCl 是否达标？地下水 Cl⁻ 是否超标？"有沉淀"答不了"有多少"。

Therefore：学习标准滴定法——用已知浓度 AgNO₃，像"化学天平"精确测量 Cl⁻ 的量。

Core Concept

滴定原理：为什么 AgNO₃ 能"称量"Cl⁻？

1. 核心反应

Ag⁺ + Cl⁻ → AgCl↓    Ksp = 1.8×10⁻¹⁰

AgCl 溶度积极小，反应几乎完全。Cl⁻ 耗尽后，Ag⁺ 与指示剂变色——终点信号。

2. 指示剂：K₂CrO₄

AgCl 的 Ksp 小于 Ag₂CrO₄。Ag⁺ 先沉淀 Cl⁻（白色），再与 CrO₄²⁻ 生成砖红色沉淀。

3. 定量关系

c(Cl⁻) = c(AgNO₃)·V(AgNO₃) / V(Cl⁻)
Procedure

滴定操作六步法

规范操作是准确结果的前提。

  1. 准备标准溶液

    基准物质配制或已标定 AgNO₃。

  2. 润洗与装液

    蒸馏水洗后，待装液润洗 2-3 次。

  3. 移取待测液

    取 25.00 mL，加 1-2 mL K₂CrO₄。

  4. 滴定操作

    左手控活塞，右手摇瓶。近终点逐滴加。

  5. 终点判断

    稳定砖红色，30 秒不褪。

  6. 读数记录

    视线水平，记录至 0.01 mL。

Calculation

滴定计算：从消耗体积到未知浓度

核心公式

c(Cl⁻) = [c(AgNO₃) × V(AgNO₃)] / V(Cl⁻)

示例

0.0500 mol/L AgNO₃ 滴定 25.00 mL NaCl，消耗 20.40 mL。

c(NaCl) = 0.0500×20.40/25.00 = 0.0408 mol/L

误差分析

失误V(AgNO₃)结果
未润洗偏大偏大
终点滞后偏大偏大
俯视读数偏小偏小
气泡未排偏大偏大

Common Mistakes

易错点诊所

这些错误在真实考试中反复出现，提前认识它们，考场上就不会掉坑。

❌ 终点判断失误

局部变红≠终点。整体稳定砖红色，30秒不褪。

❌ 单位换算遗漏

体积用升(L)，mL 乘 10⁻³。

❌ 润洗对象混淆

滴定管要润洗，锥形瓶不能润洗。

✓ 记忆口诀

管要洗，瓶莫洗；终点红，半分稳；换升算，别忘³。

Summary

本课小结

  • 原理：Ag⁺ + Cl⁻ → AgCl↓，K₂CrO₄ 指示终点
  • 操作：润洗→装液→滴定→终点→读数
  • 计算：c(Cl⁻) = c(AgNO₃)·V/V
  • 误差：终点、读数、气泡是三大来源
