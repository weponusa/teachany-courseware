# 🧬 Meiosis & Fertilization | 减数分裂与受精过程

**Subject**: Biology | **Grade**: 10 | **Type**: New Concept | **Language**: Chinese

## Overview

An interactive courseware for Grade 10 Biology covering meiosis I, meiosis II, and fertilization. Features cell division simulations and chromosome behavior visualization.

## Key Features

- 🔬 **Cell Division Simulation**: Animated canvas showing chromosome behavior through each phase
- 🧩 **Drag-and-Drop Sorting**: Students arrange meiosis phases in correct order
- 📊 **Chromosome Number Tracking**: Visual comparison of chromosome counts at each stage
- 🆚 **Mitosis vs. Meiosis Comparison**: Side-by-side interactive comparison
- 🎯 **Process Explanation Questions**: "Explain what happens to homologous chromosomes during..."

## Instructional Design

| Element | Implementation |
|:--------|:--------------|
| **ABT Narrative** | "You know cells divide by mitosis (And), but that only makes identical copies (But), sexual reproduction needs meiosis to halve the chromosomes (Therefore)" |
| **Five-Lens Method** | See (cell simulation) → Break (phase-by-phase) → Compare (mitosis vs. meiosis) |
| **Subject Framework** | Structure → Process → Function (Biology-specific) |
| **Assessment** | Diagram questions + Process explanation + Sorting exercises |
