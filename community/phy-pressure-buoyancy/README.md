# 💧 Liquid Pressure & Buoyancy | 液体压强与浮力

**Subject**: Physics | **Grade**: 8 | **Type**: New Concept | **Language**: Chinese

## Overview

An interactive courseware for Grade 8 Physics covering liquid pressure, Pascal's principle, Archimedes' principle, and buoyancy. Features virtual experiment simulations with adjustable parameters.

## Key Features

- 🧪 **Virtual Experiment Simulation**: Adjust depth, liquid density, and object size to observe pressure and buoyancy changes
- 📐 **Parameter Sliders**: Real-time formula calculation as students adjust variables
- 🔮 **Prediction-First Design**: Students predict experimental outcomes before seeing results
- 📊 **Graph Reading**: Interpret pressure-depth and force-volume relationships
- 🎯 **Three-Level Problem Solving**: Conceptual → Calculation → Design challenges

## Instructional Design

| Element | Implementation |
|:--------|:--------------|
| **ABT Narrative** | "You know objects have weight (And), but why do some float while others sink (But), understanding pressure and buoyancy explains it all (Therefore)" |
| **Five-Lens Method** | See (experiment simulation) → Break (formula derivation) → Transfer (real-world applications) |
| **Subject Framework** | Observation → Modeling → Quantitative analysis (Physics-specific) |
| **Assessment** | Prediction questions + Calculation + Experiment design |
