# 小古文里的“之乎者也” · Classical Chinese Starter

> Subject: Chinese Language · Grade: 5 · Type: New Concept · Language: Chinese

## Overview

An interactive Grade 5 Chinese courseware for introducing elementary classical Chinese reading through the high-frequency clue words “之 / 乎 / 也”. The lesson is built around three textbook passages: `杨氏之子`, `自相矛盾`, and `古人谈读书`.

## Features

- 📖 Elementary classical Chinese starter lesson aligned to `classical-chinese-intro`
- 🧩 Problem-driven structure with 4 learning modules
- 🔊 4 TTS narration segments with subtitle file
- ✍️ Record sheets for comparison, paraphrase, and transfer practice
- 🕸️ Built-in knowledge graph section linked to TeachAny node structure

## How to Use

Open `index.html` in any modern browser. No build step is required.

## Instructional Design

- **Narrative**: ABT (And-But-Therefore) + role mission
- **Assessment**: Pretest + instant feedback + posttest
- **Scaffolding**: clue-word hint → sentence comparison → transfer practice
- **Bloom's Level**: Remember → Understand → Apply

## Files

- `index.html`: main interactive courseware
- `assets/hero-classical-chinese.png`: hero illustration
- `tts/1/*.mp3`: narration audio segments
- `tts/subtitles_zh.srt`: Chinese subtitles
