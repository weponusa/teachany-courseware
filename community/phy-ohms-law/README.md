# ⚡ Ohm's Law | 欧姆定律

**Subject**: Physics | **Grade**: 9 | **Type**: New Concept | **Language**: Chinese

## Overview

An interactive courseware for Grade 9 Physics covering Ohm's Law (I = U/R), including voltage-current relationships, resistance concepts, circuit analysis, and practical applications. Features virtual experiment simulations with adjustable parameters. Includes 21 AI narration segments (~4 min).

## Key Features

- 🧪 **Virtual Circuit Lab**: Adjust voltage and resistance to observe current changes in real-time
- 📐 **Interactive Graphing**: Plot V-I characteristic curves with dynamic parameter sliders
- 🔮 **Prediction-First Design**: Students predict experimental outcomes before seeing results
- 📊 **Formula Derivation**: Step-by-step guided derivation of I = U/R
- 🎯 **Three-Level Problem Solving**: Conceptual → Calculation → Circuit design challenges
- 🔊 **AI Narration**: 21 audio segments (~4 min) for guided learning

## Instructional Design

| Element | Implementation |
|:--------|:--------------|
| **ABT Narrative** | "You know circuits carry electricity (And), but what determines how much current flows (But), Ohm's Law reveals the relationship between voltage, current, and resistance (Therefore)" |
| **Five-Lens Method** | See (circuit simulation) → Break (formula derivation) → Transfer (real-world applications) |
| **Subject Framework** | Observation → Modeling → Quantitative analysis (Physics-specific) |
| **Assessment** | Prediction questions + Calculation + Circuit design |
