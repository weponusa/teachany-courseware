# 🌍 Global Monsoon Systems | 全球季风系统

**Subject**: Geography | **Grade**: 10 | **Type**: New Concept | **Language**: Chinese

## Overview

An interactive courseware exploring global monsoon systems, covering East Asian and South Asian monsoons, seasonal wind patterns, and their impact on climate and agriculture.

## Key Features

- 🗺️ **Leaflet Interactive Map**: Pan, zoom, and explore monsoon regions on a real world map
- 💨 **Wind Pattern Visualization**: Animated arrows showing seasonal wind direction changes
- 🌡️ **Season Toggle**: Switch between summer and winter monsoon patterns
- 📊 **Regional Comparison**: Side-by-side analysis of East Asian vs. South Asian monsoons
- 🎯 **Causal Chain Exercises**: Drag-and-drop to build cause-effect relationships

## Instructional Design

| Element | Implementation |
|:--------|:--------------|
| **ABT Narrative** | "You know wind patterns exist (And), but why do they reverse with seasons (But), understanding monsoons explains climate patterns across Asia (Therefore)" |
| **Five-Lens Method** | See (map visualization) → Explain (pressure belt causes) → Compare (East vs. South Asian monsoons) |
| **Subject Framework** | Spatial distribution → Formation cause → Regional comparison (Geography-specific) |
| **Assessment** | Material analysis + Map-based questions + Regional comparison essays |
