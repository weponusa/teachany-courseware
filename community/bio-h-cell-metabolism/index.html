🎬 细胞代谢概述 · 程序化教学动画（Remotion 渲染）

📌 核心定义
细胞代谢 = 细胞内全部化学变化的总称

包括细胞内有机物的合成与分解，以及伴随发生的能量变化。

同化作用（合成代谢）

小分子 → 大分子

消耗能量，储存在有机物中

例：光合作用、蛋白质合成、DNA复制

🌿 CO₂ + H₂O → 有机物
异化作用（分解代谢）

大分子 → 小分子

释放能量，转化为ATP等形式

例：细胞呼吸、蛋白质水解、脂肪分解

⚡ 有机物 → CO₂ + H₂O + 能量

细胞代谢的两大类型：同化作用与异化作用的关系

⚗️模块二：酶——生命的催化剂

And 你已经知道细胞内每时每刻都在进行化学反应（细胞代谢）……
But 如果没有催化剂，体温37°C下的化学反应速率会极其缓慢，根本无法维持生命——那细胞是如何加速这些反应的？
Therefore 我们需要了解酶——生物体内的高效催化剂！

🎬 酶的作用机制动画（含锁钥模型/三大特性/影响因素）

🧪 酶的本质

绝大多数酶是蛋白质，少数酶是RNA（核酶）。

酶通过与底物结合形成酶-底物复合物，降低反应的活化能，从而大幅加速反应。

⚡
高效性

催化效率是无机催化剂的 10⁷ 倍

过氧化氢酶 vs Fe³⁺

🎯
专一性

一种酶只能催化一种（类）化学反应

淀粉酶不能水解蛋白质

🌈
多样性

细胞中含有成千上万种不同的酶

与基因多样性对应

酶与底物专一性结合，形成酶-底物复合物后产生产物

✏️ 解释题 — 为什么高温使酶失活？

高温会使酶永久失活，而低温只是抑制酶的活性。请解释其中的原因。（提示：思考酶的本质和空间结构）

提示：酶的活性取决于其空间结构（构象）。高温会破坏蛋白质的空间结构，这一过程称为变性（不可逆）；低温只降低分子运动速度，不破坏结构。
参考答案：酶（蛋白质）的催化活性依赖于其特定的空间结构（活性位点）。高温（如60°C以上）会使蛋白质发生变性——氢键等非共价键断裂，空间结构被永久破坏，活性位点无法与底物结合，因此酶活性丧失且不可恢复。低温（如冰箱温度）只是降低了分子的热运动速率，使酶与底物碰撞频率降低，但空间结构完整，温度升高后活性可以恢复。

🔍 额外帮助 — 酶的三大特性记忆法
特性含义实验证明

🚀 挑战延伸 — 竞争性抑制

某些药物分子的形状与底物相似，可以占据酶的活性位点但不反应，从而抑制酶的催化活性——这种现象叫做"竞争性抑制"。

思考：如果增大底物浓度，能否克服竞争性抑制？为什么？这对药物设计有何启示？

⚡模块三：ATP——细胞的能量货币

And 你已经知道细胞代谢需要能量（同化作用消耗能量，异化作用释放能量）……
But 细胞内直接使用的能量是什么形式？光合作用产生的有机物中的能量是如何被肌肉细胞利用的？
Therefore 我们需要认识ATP——细胞生命活动的直接能源物质！
🔬 ATP的结构
A — P ∼ P ∼ P
腺苷（腺嘌呤+核糖）—— 磷酸 ∼ 磷酸 ∼ 磷酸

A = 腺苷（腺嘌呤 + 核糖）

P = 磷酸基团（×3）

∼ = 高能磷酸键（含大量化学能）

ATP全名：三磷酸腺苷

含 2 个高能磷酸键

分子式：C₁₀H₁₆N₅O₁₃P₃

ATP的结构组成与ATP-ADP能量循环示意图

🔄 ATP与ADP的相互转化
ATP + H₂O ⇌ ADP + Pi + 能量
水解（放能）←→ 合成（储能）
ATP → ADP（水解）

释放能量，用于：肌肉收缩、主动运输、生物电、合成代谢...

ADP + Pi → ATP（合成）

储存能量，来自：细胞呼吸（所有生物）、光合作用（植物）

✏️ 解释题

Q5：为什么说ATP是细胞生命活动的"直接"能源物质，而葡萄糖不是？

参考答案：葡萄糖中的能量需要经过细胞呼吸才能释放出来，这个过程需要时间和多步酶促反应。而ATP高能磷酸键水解时可以直接释放能量，立即被细胞利用，中间不需要再转化。因此ATP是"直接"能源——可直接驱动各种生命活动，葡萄糖是"间接"能源。

P3 · 实验设计（解释题）

设计一个简单实验，证明酶的专一性。要求说明实验设计思路（材料、步骤、预期结果、结论）。

参考方案：
材料：淀粉溶液、蛋白质溶液（如蛋清稀释液）、淀粉酶液、碘液、双缩脲试剂
分组：实验组：淀粉溶液+淀粉酶 / 对照组：蛋白质溶液+淀粉酶
步骤：37°C水浴适当时间后，各管分别检测（碘液/双缩脲）
预期结果：实验组碘液不变色（淀粉已分解），双缩脲阳性（蛋白质未变化）；对照组碘液变蓝（淀粉未分解）
结论：淀粉酶只能催化淀粉水解，不能催化蛋白质水解，体现酶的专一性。
