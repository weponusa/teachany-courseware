## 教学目标

1. 了解细胞的发现历史，认识科学家在细胞学说建立过程中的贡献
2. 理解"细胞是生命活动的基本单位"这一核心概念
3. 初步区分动物细胞和植物细胞的异同点
4. 能用显微镜观察并辨认细胞结构（与前置课衔接）

## 问题锚点设计

**核心驱动问题**：所有生物体都由细胞构成吗？一滴水中藏着怎样的微观世界？

**三个切入选项**：
1. "为什么说细胞是生命的基本单位？"——直击核心概念
2. "植物细胞和动物细胞看起来一样吗？"——对比认知
3. "第一个看到细胞的人看到了什么？"——历史兴趣切入

## ABT 叙事设计

### Module 1: 细胞的发现历史
- **And**（已知）：我们知道生物体有各种各样的形态和功能
- **But**（冲突）：肉眼看不到组成生物体的最小单元是什么
- **Therefore**（目标）：了解科学家如何借助显微镜发现并命名了"细胞"

### Module 2: 细胞是生命活动的基本单位
- **And**（已知）：显微镜下能看到各种各样的细胞
- **But**（冲突）：为什么说所有生命活动都离不开细胞？
- **Therefore**（目标）：理解细胞与生命活动的关系

### Module 3: 动植物细胞的区别概述
- **And**（已知）：动物和植物都由细胞组成
- **But**（冲突）：它们的细胞结构一样吗？
- **Therefore**（目标）：学会辨别动植物细胞的结构差异

## 互动设计

### Canvas 互动：虚拟显微镜观察
- 学生拖动滑块调节放大倍数（40x → 100x → 400x）
- 画布实时绘制不同放大倍率下的洋葱表皮细胞
- 可切换观察对象（洋葱表皮 / 口腔上皮）

### 外部工具嵌入
- PhET 基因表达模拟（用于展示细胞功能）
- 3Dmol.js 展示细胞内蛋白质结构

## 评估设计

### 前测
- 检测学生对"生物体基本组成"的朴素认知
- 识别常见误概念：把组织当作最小单位

### ConcepTest
- 概念检测：细胞学说的核心内容
- 干扰项设计对应常见误解

### 后测
- 新情境应用：判断某结构是否为细胞
- 区分动植物细胞的关键特征

## 教学流程

1. **问题锚点** → 激发动机（2分钟）
2. **前测** → 暴露前概念（3分钟）
3. **模块1：发现历史** → 建立背景知识（5分钟）
4. **模块2：基本单位** → 核心概念讲解 + Canvas互动（8分钟）
5. **ConcepTest** → 即时检验理解（3分钟）
6. **模块3：动植物对比** → 结构对比 + 外部工具互动（7分钟）
7. **后测** → 检验迁移能力（3分钟）
8. **小结** → 回答锚点问题，连接后续课程（2分钟）
