# 📐 Quadratic Functions | 二次函数 y=ax²+bx+c

**Subject**: Math | **Grade**: 9 | **Type**: New Concept | **Language**: Chinese

## Overview

A complete interactive courseware for Grade 9 students learning quadratic functions. Covers the general form, vertex form, completing the square, axis of symmetry, and graph transformations.

## Key Features

- 🎨 **Canvas Interactive Graphing**: Students drag the vertex to see the equation change in real-time
- 📐 **Step-by-Step Derivation**: Completing the square broken into manageable steps with animation
- 🔍 **Error Diagnosis**: Per-option feedback explains exactly what went wrong (e.g., "You flipped the sign of h")
- 🎯 **Three-Level Exercises**: Foundation → Application → Transfer Challenge
- 📊 **Pre-test & Post-test**: Full learning loop with progress tracking

## Instructional Design

| Element | Implementation |
|:--------|:--------------|
| **ABT Narrative** | "You can plot quadratics (And), but point-by-point is slow (But), completing the square reveals the vertex instantly (Therefore)" |
| **Five-Lens Method** | See (translation animation) → Break (completing the square process) → Transfer (general formula) |
| **Scaffolding** | Level 1: Read vertex → Level 2: Complete the square → Level 3: Design a parabola |
| **Bloom's Coverage** | Remember, Understand, Apply, Analyze, Evaluate, Create |
