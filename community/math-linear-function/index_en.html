🤔 Starting from Real-Life Problems

[And] You already know what a function is, how the coordinate plane works, and that y changes with x — for example, distance traveled increases with time.
[But] In real life, many "constant-rate" changes — phone bills, taxi fares, spring stretching — all form straight lines, and a generic function description is too vague.
[Therefore] So we need to learn the simplest type of function — the linear function y = kx + b. Its graph is a straight line, and just two parameters describe all constant-rate phenomena.

📊 Lesson 1: Proportional Function y = kx

📌 Introduction

Spring experiment: The extension y (cm) of a spring vs. the mass x (kg) of the hanging weight:

x / kg01234
y / cm02468

Observation: y = 2x — y is directly proportional to x.

📝 Definition

In general, a function of the form y = kx (k is a constant, k ≠ 0) is called a proportional function (direct variation).

Here, k is called the constant of proportionality.

🔍 Deep Understanding: Why Does the Graph Always Pass Through the Origin?

Because in y = kx, when x = 0, y = k·0 = 0, so the point (0, 0) is always on the graph. This is the essence of "direct proportion" — there is no initial quantity; everything starts from zero. A spring with no weight has zero extension; zero minutes of travel means zero distance.

🎬 Proportional Function y = kx — Graphing

📐 Graph Features

1
The graph is a straight line that always passes through the origin (0, 0)
2
When k > 0: passes through Quadrants I and III; y increases as x increases
3
When k < 0: passes through Quadrants II and IV; y decreases as x increases

k > 0I, IIIIncreasing ↗
k < 0II, IVDecreasing ↘

📈 Lesson 2: Linear Function y = kx + b

📌 From Proportional to Linear

A phone plan charges a monthly fee of $10 plus $0.30 per minute. If you talk for x minutes, the monthly cost y is:

y = 0.3x + 10

This function has an extra constant term +10 compared to a proportional function — this is a linear function.

📝 Definition

In general, a function of the form y = kx + b (k, b are constants, k ≠ 0) is called a linear function.

When b = 0, y = kx is a proportional function. So a proportional function is a special case of a linear function.

🔍 Deep Understanding: What Is the Essence of b?

b represents the "initial value" or "starting point" — the monthly base fee, the taxi base fare, the initial water in a tank... In real life, many changes don't start from zero, and b represents that baseline. On the graph, b determines where the line crosses the y-axis at (0, b). Changing b shifts the entire line up or down.

🎬 Linear Function y = kx + b — Introduction

🖊️ Graphing Method: Two-Point Method

The graph of a linear function is a straight line. Two points determine a line, so we only need two special points:

1
Set x = 0: y = b → Point (0, b) (y-intercept)
2
Set y = 0: x = −b/k → Point (−b/k, 0) (x-intercept)
3
Draw a straight line through these two points

🔍 Lesson 3: Slope, Intercept & Graph Properties

📈 Geometric Meaning of Slope k
🔀 Translation Effect of Intercept b

📐 Meaning of Slope k

k = Δy / Δx = (y₂ − y₁) / (x₂ − x₁)
  • Larger |k| → steeper line
  • k > 0: rises from left to right ↗
  • k < 0: falls from left to right ↘
  • k = 1: 45° angle with the x-axis

🔀 Meaning of Intercept b

b is the y-coordinate where the line crosses the y-axis, i.e., the point (0, b)

  • b > 0: line shifts up
  • b = 0: passes through origin (becomes a proportional function)
  • b < 0: line shifts down

💡 Changing b is like translating y = kx vertically by |b| units

🗺️ Signs of k and b Determine Which Quadrants the Graph Passes Through

ConditionQuadrantsApproximate Shape
k > 0, b > 0I, II, III↗ Rising, crosses positive y-axis
k > 0, b < 0I, III, IV↗ Rising, crosses negative y-axis
k < 0, b > 0I, II, IV↘ Falling, crosses positive y-axis
k < 0, b < 0II, III, IV↘ Falling, crosses negative y-axis

💡 Memory Aid: k positive = rises left-to-right; k negative = falls left-to-right; b positive = shifts up; b negative = shifts down

✍️ Worked Examples

📝 Example 1: Identifying Proportional Functions

Which of the following are proportional functions? Which are linear functions?

① y = 3x    ② y = 2x + 1    ③ y = x²    ④ y = −5x    ⑤ y = 3/x    ⑥ y = −2x + 3

👁️ Show Solution
✅ Solution

Proportional functions (form y = kx, k ≠ 0): ① y = 3x, ④ y = −5x

Linear functions (form y = kx + b, k ≠ 0): ① ② ④ ⑥

③ y = x² is a quadratic function, not linear
⑤ y = 3/x is an inverse proportional function, not linear
Every proportional function is a linear function, but not every linear function is proportional

📝 Example 2: Finding the Equation (Undetermined Coefficients)

A linear function y = kx + b passes through A(1, 5) and B(−1, 1). Find the equation.

👁️ Show Solution
✅ Solution (Method of Undetermined Coefficients)
1
Substitute A(1, 5) into y = kx + b: k + b = 5 … ①
2
Substitute B(−1, 1) into y = kx + b: −k + b = 1 … ②
3
① + ②: 2b = 6, so b = 3
4
Substitute into ①: k = 5 − 3 = 2

∴ The linear function is y = 2x + 3

📝 Example 3: Graph & Properties

For the linear function y = −2x + 4:

(1) Where does it intersect the x-axis and y-axis?

(2) Which quadrants does it pass through?

(3) For what values of x is y > 0?

👁️ Show Solution
✅ Solution

(1)

  Set x = 0: y = 4 → y-intercept is (0, 4)

  Set y = 0: −2x + 4 = 0 → x = 2 → x-intercept is (2, 0)

(2)

  k = −2 < 0, b = 4 > 0 → passes through Quadrants I, II, IV

(3)

  y > 0 means −2x + 4 > 0 → x < 2

  ∴ When x < 2, y > 0

💡 This connects linear functions to linear inequalities! The part of the graph above the x-axis corresponds to y > 0

📝 Example 4: Real-World Modeling

A water tank initially contains 20 liters. Water flows in at 5 liters per hour.

(1) Write the equation for water volume y (liters) vs. time x (hours)

(2) How much water after 8 hours?

(3) The tank holds at most 80 liters. When is it full?

👁️ Show Solution
✅ Solution

(1) y = 5x + 20 (x ≥ 0)

(2) When x = 8: y = 5 × 8 + 20 = 60 liters

(3) When y = 80: 80 = 5x + 20 → x = 12 hours

🚀 Challenge Problems

🏆 Challenge 1: Intersection of Two Lines

Given y₁ = 2x − 1 and y₂ = −x + 5, find their intersection and determine for which x values y₁ > y₂.

👁️ Show Solution
✅ Solution

Intersection: Set 2x − 1 = −x + 5

  3x = 6 → x = 2 → y = 2(2) − 1 = 3

  ∴ Intersection: (2, 3)

Find y₁ > y₂:

  2x − 1 > −x + 5 → 3x > 6 → x > 2

💡 Graphically: When x > 2, the line y₁ = 2x − 1 is above y₂ = −x + 5

🏆 Challenge 2: Area Calculation

Find the area of the triangle formed by y = −¾x + 3 and the coordinate axes.

👁️ Show Solution
✅ Solution

Set x = 0: y = 3 → y-intercept A(0, 3)

Set y = 0: −¾x + 3 = 0 → x = 4 → x-intercept B(4, 0)

Base = OB = 4, Height = OA = 3

Area = ½ × 4 × 3 = 6

🏆 Challenge 3: Parameter Discussion

The linear function y = (m−2)x + (m+1) does not pass through Quadrant II. Find the range of m.

👁️ Show Solution
✅ Solution

First, for it to be a linear function: k ≠ 0, so m − 2 ≠ 0 → m ≠ 2

To avoid Quadrant II, we need:

  In Quadrant II: x < 0, y > 0

  Need k > 0 and b ≤ 0 (graph passes through Quadrants I, III, IV or I, III)

That gives us:

  m − 2 > 0 → m > 2

  m + 1 ≤ 0 → m ≤ −1

These two conditions cannot both be satisfied simultaneously.

⚠️ If the problem were "does not pass through Quadrant III": k > 0 and b > 0 → m > 2 and m > −1 → m > 2

🏆 Challenge 4: Comprehensive Problem

A linear function y = kx + b passes through A(−3, 4) and B(2, −1).

(1) Find the equation; (2) Find the area of the triangle formed with the axes; (3) Directly write the solution set for kx + b > 0.

👁️ Show Solution
✅ Solution

(1) Substitute A(−3, 4): −3k + b = 4 … ①

  Substitute B(2, −1): 2k + b = −1 … ②

  ② − ①: 5k = −5 → k = −1 → b = 1

  ∴ y = −x + 1

(2)

  y-intercept: (0, 1); x-intercept: (1, 0)

  Area = ½ × 1 × 1 = ½

(3)

  kx + b > 0 means −x + 1 > 0 → y > 0

  The part of the graph above the x-axis corresponds to x < 1

  ∴ Solution set: x < 1
