# 📏 Linear Functions | 一次函数与正比例函数

**Subject**: Math | **Grade**: 8 | **Type**: New Concept | **Language**: Chinese & English

## Overview

A comprehensive interactive courseware covering proportional functions and linear functions for Grade 8 students. Includes slope, y-intercept, graph drawing, and real-world applications.

## Key Features

- 📈 **Real-Time Graphing**: Adjust slope (k) and intercept (b) sliders to see the graph update instantly
- 🔢 **Step-by-Step Reasoning**: From proportional to linear functions with smooth transitions
- 🎯 **ConcepTest Questions**: Designed for 30-70% accuracy — ideal for peer discussion
- ✏️ **Error Diagnosis**: Specific feedback for common mistakes (slope direction, intercept confusion)
- 📊 **Three-Level Exercises**: Progressive difficulty with scaffolding

## Instructional Design

| Element | Implementation |
|:--------|:--------------|
| **ABT Narrative** | "You know y=kx passes through the origin (And), but many real-world lines don't (But), so we add +b to get y=kx+b (Therefore)" |
| **Five-Lens Method** | See (graph visualization) → Break (slope formula step-by-step) → Compare (proportional vs. linear) |
| **Cognitive Load** | One concept per card, ~75 words, immediate practice after each concept |
