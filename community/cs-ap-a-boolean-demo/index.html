📖 💻 逻辑运算符真值表

ABT 引入

已知：你会写 int、String 变量和算术运算

问题：但程序要做决策——怎么让它[判断]？

解法：布尔值（true/false）+ 逻辑运算符（&&、||、!）= 程序决策的基础

核心步骤

&&（AND）：两个都为 true → true；否则 false
||（OR）：至少一个为 true → true；两个都 false → false
!（NOT）：取反，true → false，false → true
De Morgan 定律：!(A && B) == !A || !B
if (条件) { 执行A } else { 执行B }

⚠️ 易错点

易错 1
= vs ==：赋值与比较混用
✅ = 是赋值；== 是比较；equals() 比较字符串
易错 2
短路求值：&&左边false时右边不执行
✅ 利用短路：先判安全条件再判具体条件

📌 小结

  • &&：全真才真；||：有真即真；!：取反
  • De Morgan: !(A&&B) = !A||!B
  • if-else 是条件判断的基本结构
