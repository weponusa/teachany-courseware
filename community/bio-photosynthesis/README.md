# 🧬 Photosynthesis | 光合作用

**Subject**: Biology | **Grade**: 7 | **Type**: New Concept | **Language**: Chinese

## Overview

An interactive courseware for Grade 7 Biology covering the definition of photosynthesis, reaction equation, three essential conditions, chloroplast structure, and photosynthesis vs. respiration comparison. Features Canvas animations, drag-and-drop sorting, instant feedback, and error diagnosis. Includes 12 AI narration segments with subtitles.

## Key Features

- 🎨 **Canvas Animation**: Animated visualization of the photosynthesis process inside chloroplasts
- 🧩 **Drag-and-Drop Sorting**: Students arrange reaction equation components in correct order
- 🔬 **Chloroplast Structure**: Interactive labeling of chloroplast anatomy
- 🆚 **Photosynthesis vs. Respiration**: Side-by-side comparison with interactive matching
- 🎯 **Error Diagnosis**: Per-option feedback explaining common misconceptions
- 🔊 **AI Narration**: 12 audio segments (~3 min) with synchronized subtitles

## Instructional Design

| Element | Implementation |
|:--------|:--------------|
| **ABT Narrative** | "Plants need food to survive (And), but they don't eat like animals (But), they make their own food through photosynthesis (Therefore)" |
| **Five-Lens Method** | See (Canvas animation) → Break (equation components) → Compare (photosynthesis vs. respiration) |
| **Subject Framework** | Structure → Process → Function (Biology-specific) |
| **Assessment** | Pre-test + Drag-sort exercises + Process explanation + Error diagnosis |
