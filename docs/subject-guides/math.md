# Math Subject Guide | 数学学科指南

## Teaching Approach

**Visual Intuition + Algebraic Reasoning + Generalization**

Math courses should always follow this three-step pattern:
1. **Start with visual intuition** — Let students see the concept graphically before introducing formulas
2. **Build algebraic reasoning** — Walk through the derivation step by step
3. **Generalize** — Show that the specific example represents a general pattern

## Best Interaction Types

| Interaction | When to Use | Example |
|:------------|:-----------|:--------|
| **Canvas graphing** | Function visualization | Plot y=ax²+bx+c, drag parameters |
| **Parameter sliders** | Exploring how variables affect results | Adjust a, b, c and see the graph change |
| **Step-by-step derivation** | Complex calculations | Completing the square, one step at a time |
| **Error diagnosis** | Common calculation mistakes | "You flipped the sign of h" |

## Five-Lens Preferences

For Math, the most effective lens combinations are:

- **"Why does this work?"** → See (graph) + Break (derivation) + Transfer (general form)
- **"I keep getting it wrong"** → Compare (correct vs. incorrect) + Break (step-by-step)
- **"What's the point?"** → See (real-world application) + Transfer (new problem)

## Common Errors to Diagnose

| Topic | Common Error | Diagnosis |
|:------|:------------|:----------|
| Quadratic vertex form | Writing h=-3 for (x-3) | "The minus sign is already in the formula: y=a(x-h)². So x-3 means h=3" |
| Slope | Confusing Δy/Δx with Δx/Δy | "Slope is rise over run — vertical change divided by horizontal change" |
| Completing the square | Forgetting to multiply by a | "When a≠1, you must factor out a first, then complete the square inside" |

## Assessment Mix

- **Objective**: 50% (calculation, identification)
- **Explanation**: 30% ("Why is the axis of symmetry at x=h?")
- **Production**: 20% ("Design a parabola passing through these points")
