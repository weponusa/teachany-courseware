# Physics Subject Guide | 物理学科指南

## Teaching Approach

**Observation + Modeling + Quantitative Analysis**

Physics courses should follow this sequence:
1. **Start with a phenomenon** — Show a surprising or counter-intuitive observation
2. **Build a model** — Help students construct a mental model that explains the phenomenon
3. **Quantify** — Introduce formulas as a way to make predictions from the model

## Best Interaction Types

| Interaction | When to Use | Example |
|:------------|:-----------|:--------|
| **Parameter sliders** | Exploring physical relationships | Adjust depth, density, see pressure change |
| **Experiment simulation** | Virtual lab experience | Drop objects in different liquids |
| **Prediction cards** | "What do you think will happen?" | Predict before revealing results |
| **Graph reading** | Interpreting data | Pressure vs. depth graph |

## Five-Lens Preferences

- **"Why does this happen?"** → See (experiment) + Explain (model) + Transfer (prediction)
- **"I don't understand the formula"** → Break (derivation) + See (visual representation)
- **"When does this apply?"** → Compare (different scenarios) + Transfer (new problem)

## Assessment Mix

- **Prediction**: 30% ("What happens if we double the depth?")
- **Calculation**: 30% (Apply formula to solve)
- **Explanation**: 40% ("Why does the object float in salt water but sink in fresh water?")
