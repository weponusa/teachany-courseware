# History Subject Guide | 历史学科指南

## Teaching Approach

**Chronological Thread → Causal Chain → Source Evidence / Multi-Perspective**

History courses should follow this framework:
1. **Chronological Thread** — When did it happen? What's the sequence?
2. **Causal Chain** — Why did it happen? What factors led to this?
3. **Source Evidence** — What's the evidence? How do we know?
4. **Multi-Perspective** — How did different groups see this? Whose voice is missing?

## Best Interaction Types

| Interaction | When to Use | Example |
|:------------|:-----------|:--------|
| **Timeline sorting** | Establishing chronology | Arrange reform events in order |
| **Source comparison** | Evaluating evidence | Read two accounts of the same event |
| **Stance analysis** | Understanding perspectives | "Which group would support this reform?" |
| **Causal chain drag-and-drop** | Building cause-effect understanding | Connect background → policy → outcome |
| **Comparison columns** | Analyzing different reforms/events | Side-by-side: Shang Yang vs. Wu Qi |

## Five-Lens Preferences

- **"When did this happen?"** → See (timeline) + Break (phases)
- **"Why did this happen?"** → Explain (causal chain) + Compare (different factors)
- **"How do we know?"** → See (primary sources) + Compare (different accounts)
- **"Why did the reform fail/succeed?"** → Compare (successful vs. failed) + Explain (key conditions)

## Assessment Mix

- **Objective**: 30% (chronology, factual recall)
- **Source-Based**: 30% ("Which source is more reliable? Why?")
- **Essay/Production**: 40% ("Write a 200-word analysis citing at least one source")

## Scaffolding for Essays

| Level | Support | Example |
|:------|:--------|:--------|
| Level 1 | Argument + evidence framework provided | "Fill in the evidence for each argument" |
| Level 2 | Argument prompts only | "Support this claim with evidence" |
| Level 3 | Topic only | "Analyze the key factors in 200 words" |

## Rubric for Open Tasks

| Dimension | Weight | Description |
|:----------|:-------|:-----------|
| Content Accuracy | 25% | Facts correct, time periods accurate |
| Evidence Sufficiency | 30% | Cites primary/secondary sources |
| Structural Clarity | 25% | Logical argument structure |
| Expression Quality | 20% | Appropriate academic register |
