## 四、Fork 项目的命名要求 Forking Naming Rules

AGPL-3.0 允许任何人 fork 并修改 TeachAny 代码，但**修改后的衍生版本必须改名**：

### ✅ 推荐的改名方式

| 原名 | ❌ 不允许 | ✅ 合规 |
|:---|:---|:---|
| TeachAny → | TeachAny Enhanced | OpenLearn-EDU |
| TeachAny → | TeachAny Pro | NextCourse |
| 教我学 → | 教我学 Plus | 智学家园 |
| 教我学 → | 新教我学 | 启慧课件 |

核心原则：**不含 "TeachAny" / "教我学" / "教我" / "TeachA" 等任何连续 4 个字符**。

### ✅ 允许的声明方式

在你的 fork 项目 README 里声明：

```markdown
## Credits

This project is a fork of [TeachAny](https://github.com/weponusa/teachany) (AGPL-3.0).
Modifications: [summary of your changes]
```

不允许：

```markdown
## TeachAny Community Edition   ← 禁止用此标题
```
